# Weather Background Images

This folder contains high-resolution weather-themed background images for the Maya AI Weather Assistant.

## Required Images

Place the following images in this directory (recommended resolution: 1920x1080 or higher):

- `default.jpg` - Default background (clear sky or gradient)
- `sunny.jpg` - Bright sunny day with blue sky
- `cloudy.jpg` - Overcast cloudy sky
- `rainy.jpg` - Rain or storm scene
- `foggy.jpg` - Misty or foggy atmosphere
- `stormy.jpg` - Dramatic storm or thunder clouds

## Image Sources

You can get free high-quality weather images from:

- Unsplash.com (search: "weather", "sky", "clouds", "rain")
- Pexels.com (royalty-free weather photos)
- Pixabay.com (free weather backgrounds)

## Usage

These images are automatically loaded based on current weather conditions:

- Sunny/Clear → sunny.jpg
- Clouds → cloudy.jpg
- Rain → rainy.jpg
- Mist/Fog → foggy.jpg
- Storm/Thunder → stormy.jpg
- Default → default.jpg

The images are displayed with 30% opacity and blur overlay for optimal text readability.
